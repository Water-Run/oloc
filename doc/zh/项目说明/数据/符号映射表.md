# 符号映射表  

在`oloc`,对于下表右边的符号,将在与处理中转化为左边形式.  
处理过程依照表格顺序进行:  

| 名称     | 转化结果 | 原始符号                                          | 备注              |
|--------|------|-----------------------------------------------|-----------------|
| 空格     | (无)  | ` `                                           | 空格被替换为空         | 
| 无效符号   | (无)  | `_` `=` `equal` `equals` `is` `rad` `radians` | 不影响计算表达式的符号     |
| 指数符号   | `^`  | `^` `**`                                      | 无               |
| 加号(正号) | `+`  | `+` `plus`                                    | 无               |  
| 减号(负号) | `-`  | `-` `minus`                                   | 无               |  

***符号映射表的自定义扩展***  
以上内容是`oloc`的原生符号映射表.你也可以通过重写,实现自定义的符号映射表进行扩展,甚至可以实现对近乎自然语言输入的支持.
