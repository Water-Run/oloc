# 符号映射表  

在`oloc`,对于下表右边的符号,将在与处理中转化为左边形式.  
处理过程依照表格顺序进行:  

| 名称     | 转化结果 | 原始符号                      | 备注              |
|--------|------|---------------------------|-----------------|
| 空格     | (无)  | ` `                       | 空格被替换为空         | 
| 等于号    | (无)  | `=` `equal` `equals` `is` | 等于号被替换为空        |
| 弧度符号   | (无)  | `rad` `radians`           | 默认情况下即为弧度制,替换为空 |
| 指数符号   | `^`  | `^` `**`                  | 无               |
| 加号(正号) | `+`  | `+` `plus`                | 无               |  
| 减号(负号) | `-`  | `-` `minus`               | 无               |

***符号映射表的自定义扩展***  
以上内容是`oloc`的原生符号映射表.你也可以通过重写,实现自定义的符号映射表进行扩展,甚至可以实现对近乎自然语言输入的支持.
