# 符号映射表  

在`oloc`,对于下表右边的符号,将在与处理中转化为左边形式.  
处理过程依照表格顺序进行:  

| 名称     | 转化结果 | 原始符号                      | 备注         |
|--------|------|---------------------------|------------|
| 空格     | (无)  | ` `                       | 空格被替换为空    | 
| 等于号    | (无)  | `=` `equal` `equals` `is` | 仅当位于表达式结尾时 |
| 指数符号   | `^`  | `^` `**`                  | 无          |
| 加号(正号) | `+`  | `+` `plus`                | 无          |  
| 减号(负号) | `-`  | `-` `minus`               | 无          |
